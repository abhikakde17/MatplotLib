# Online Retail Data Analysis

## Overview
This project focuses on performing Exploratory Data Analysis (EDA) on an online retail dataset to understand sales patterns, customer behavior, and revenue distribution.

## Dataset
- Online Retail Dataset
- Records: 530,000+ transactions
- Data includes invoice details, product information, quantity, price, customer ID, and country.

## Objectives
- Clean and preprocess raw retail data
- Analyze sales trends and customer behavior
- Identify top-selling products and revenue patterns
- Visualize insights using Python libraries

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## Key Steps
- Removed missing values, duplicates, and invalid transactions
- Performed Exploratory Data Analysis (EDA)
- Analyzed sales trends and product performance
- Visualized country-wise revenue and overall sales patterns

## Results
- Identified top-selling products
- Analyzed revenue distribution across 38 countries
- Gained insights into customer purchasing behavior

## Conclusion
This analysis helped in understanding key sales trends and provided actionable insights using data-driven techniques.
